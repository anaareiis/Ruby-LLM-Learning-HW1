# Ruby + LLM Study – Homework 01

## 📌 Objective

This project explores how Large Language Models (LLMs) can support learning and development in Ruby through the resolution of the Ruby Calisthenics Homework 01.

## 🧠 About the Project

All exercises from Parts 01 to 06 were implemented and analyzed with the assistance of LLM tools. The goal is not only to solve the problems but also to understand:

* Ruby programming concepts
* Different implementation approaches
* Trade-offs in design decisions

## 📂 Structure

Each file corresponds to a part of the assignment:

* Part 1: String manipulation
* Part 2: Rock-Paper-Scissors
* Part 3: Anagrams
* Part 4: Basic OOP
* Part 5: Metaprogramming
* Part 6: Advanced Ruby features

## 🤖 Role of LLM

LLMs were used to:

* Clarify concepts
* Suggest improvements
* Provide alternative implementations
* Support debugging
