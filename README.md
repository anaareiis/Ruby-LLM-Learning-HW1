# 📚 Ruby + LLM Study – Homework 01

## 🎯 Objective

This project explores how Large Language Models (LLMs) can support learning and the development of Ruby applications through the resolution of **Ruby Calisthenics – Homework 01**.

---

## 🧠 About the Project

This project was developed as part of **Homework 01 for the Software Engineering course at the University of Brasília (UnB)**.

All exercises from Parts 01 to 06 were fully implemented and critically analyzed with the support of LLM tools.

The main goal is not only to solve the problems, but also to understand:

* Ruby programming concepts
* Different implementation approaches
* Trade-offs in design decisions

---

## 📂 Project Structure

Each file corresponds to a part of the assignment:

* **Part 1:** String manipulation
* **Part 2:** Rock-Paper-Scissors
* **Part 3:** Anagrams
* **Part 4:** Basic OOP
* **Part 5:** Metaprogramming
* **Part 6:** Advanced Ruby features

---

## 🤖 Role of LLMs

LLMs were used as learning assistants throughout the project to:

* Clarify concepts
* Suggest improvements
* Provide alternative implementations
* Support debugging and error analysis
